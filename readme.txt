Порядок установки сборки:
1) установить node.js
   ссылка для установки: https://nodejs.org/en/
2) создать пустой каталог(папку) в произвольном месте.
   перейти в него из командной строки и склонировать
   репозиторий по ссылке               https://github.com/moroz-igor/Gulp-Frontend.git
   команда для клонирования: git clone https://github.com/moroz-igor/Gulp-Frontend.git
3) после клонирования репозитория небходимо перейти в папку 'gulp'
   из командной строки. Далее выполнить команды в командной строке:
	npm install gulpjs/gulp-cli -g     // дождаться окончания установки
	gulp -v
				отклик дожен быть:
				                    [15:27:06] CLI version 1.4.0
	npm install gulpjs/gulp#4.0 --save-dev   //дождаться окончания установки (занимает несколько минут)
						 // в каталог проэкта установится папка node_modules
	gulp -v
				отклик дожен быть:
									[15:27:06] CLI version 1.4.0
									[15:27:07] Local version 4.0.0-alpha.2
	если всё так, запускаем сборку командой:

4)  gulp
    должны запуститься все плагины  и gulp watchers,
    должено открыться окно браузера со стартовой страницей сборки,
	выключить gulp watchers в командной строке можно нажав 'ctrl' + 'c'

P.S. Наа момент установки сборки версии окружения могут отличаться...
